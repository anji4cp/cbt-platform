# CBT Platform

Aplikasi Computer Based Test (CBT) multi sekolah berbasis web.

## Deskripsi
CBT Platform adalah aplikasi ujian berbasis web yang dirancang untuk digunakan oleh banyak sekolah.
Sistem mendukung manajemen sekolah, admin sekolah, guru, dan siswa.

## Fitur Utama
- Multi sekolah
- Manajemen admin sekolah
- Manajemen siswa dan kelas
- Ujian berbasis PDF (Google Drive)
- Token ujian
- Sistem submit ujian
- Pengaturan durasi dan minimal waktu submit
- Penilaian otomatis

## Teknologi
- Laravel
- PHP
- PostgreSQL / MySQL
- Tailwind CSS

## Status
Masih dalam pengembangan (MVP)

## Catatan
Project ini dikembangkan secara mandiri sebagai produk nyata dan portofolio.
